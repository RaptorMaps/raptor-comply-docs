This guide assumes you already have an API key and your organization slug. If not, see [Authentication](/authentication).

## Make your first call

List the facilities in your organization:

```bash
curl https://api.raptormaps.com/facilities \
  -H "X-API-Key: $RAPTOR_COMPLY_API_KEY" \
  -H "Organization: $RAPTOR_COMPLY_ORG_SLUG"
```

A successful response returns a JSON array of facilities you have access to:

```json
[
  {
    "id": "fac_01HXY...",
    "name": "South Plant Substation",
    "...": "..."
  },
  {
    "id": "fac_01HXZ...",
    "name": "North Field Switching Station",
    "...": "..."
  }
]
```

## Node.js example

```javascript
const response = await fetch("https://api.raptormaps.com/facilities", {
  headers: {
    "X-API-Key": process.env.RAPTOR_COMPLY_API_KEY,
    "Organization": process.env.RAPTOR_COMPLY_ORG_SLUG,
  },
});

if (!response.ok) {
  const error = await response.json();
  throw new Error(`${error.error}: ${error.message}`);
}

const facilities = await response.json();
console.log(facilities);
```

## Python example

```python
import os
import requests

response = requests.get(
    "https://api.raptormaps.com/facilities",
    headers={
        "X-API-Key": os.environ["RAPTOR_COMPLY_API_KEY"],
        "Organization": os.environ["RAPTOR_COMPLY_ORG_SLUG"],
    },
)
response.raise_for_status()
facilities = response.json()
```
