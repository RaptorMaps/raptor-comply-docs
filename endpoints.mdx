<Note>
  Availability is governed by OpenFGA. Endpoints marked **Available** today
  accept your API key on `api.raptormaps.com`. Endpoints marked **Coming soon**
  exist in our internal API but are not yet exposed via API-key auth. We're
  rolling them out feature-by-feature. Subscribe to release notes (link
  forthcoming) to be notified as each batch lands.
</Note>
